Projet fin de 1ére année

Site web

Juin 2021

Présentation

Ce projet à était réaliser pour valider la 1ère année. Le sujet était de concevoir un site web sur une personnage de fiction.

Il fallait y intéger une galérie d’image, un blog ainsi qu’une gestion des utilisateurs pour que certain puisse écrire des articles de blog.

Technologie utiliser

Pour réaliser ce site web, il y avait des contraintes de technologie.

En front-end le Html, CSS, ainsi que le JS était autoriser. Une seule librairie JS était autoriser, le JQuery.

Pour le back-end, PHP et MySql ont était utiliser pour la gestion de base de données, lier avec PhpMyAdmin.
