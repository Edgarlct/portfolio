Présentation

Ce projet est un projet personnel. Il s’agit d’un robot de trading de crypto monaie. Le robot de base sur différente donnée récupérer via l’api du site Binance. Avec ces indicateur le robot essaie de savoir quand acheter et vendre.

Ce robot, est déjà fonctionnel mais il nécessite des ajustements de configuration pour intégrer dans sa stratégie les frais appliquer par la plateforme.

Technologie utiliser

Pour réaliser cette algorythme 2 technologie on était utilisé.

Pour la récupération des données (prix, quantité vendu ... ), tout passe par l’api de Binance qui permet également l’envoie d’ordre d’achat et de vente

Pour l’analyse et les appels à l’api, tout ce fait avec du python.
